1️⃣  GET FREE API KEY

    → Sign up (free) and copy your API key

2️⃣  UPDATE CODE
    → Open: app/build.gradle
    → Find: buildConfigField "String", "NEWS_API_KEY"
    → Replace dummy key with your real key
    → Save

3️⃣  RUN APP
    → Open Android Studio
    → File → Open → Select CyberSecurityNews folder
    → Click green Run button
    → Select emulator or device
    → Done! 🎉

"Failed to load news"
└─ Check internet connection
└─ Verify API key is correct
└─ Check API quota (100 requests/day limit)

"Gradle sync failed"
└─ File → Sync Now
└─ File → Invalidate Caches / Restart

"App crashes"
└─ Ensure Android API ≥21
└─ Check API key in build.gradle
└─ Verify all files saved
